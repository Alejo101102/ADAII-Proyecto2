=== PROYECTO MINEXT - README ===

ARCHIVOS ENTREGADOS:
1. Readme.txt - Este archivo con instrucciones
2. Informe.pdf - Informe completo del proyecto con modelo y análisis
3. Proyecto.mzn - Implementación del modelo MiniZinc
4. DatosProyecto/ - Directorio con datos de prueba
5. ProyectoGUIFuentes/ - Archivos fuente de la interfaz gráfica
6. MisInstancias/ - 5 instancias generadas por el equipo

INSTRUCCIONES PARA EJECUTAR LA APLICACIÓN:

REQUISITOS:
- Python 3.7 o superior
- MiniZinc instalado y accesible desde línea de comandos

PASOS PARA EJECUTAR:
1. Abrir terminal o línea de comandos
2. Navegar al directorio ProyectoGUIFuentes/
3. Ejecutar: python interfaz.py

USO DE LA APLICACIÓN:
1. Seleccionar archivo de entrada (.txt)
2. Seleccionar archivo del modelo (Proyecto.mzn)
3. Configurar parámetros si es necesario (solver)
4. Presionar "Ejecutar Modelo MinExt"
5. Ver resultados en las pestañas disponibles

FUNCIONALIDADES ADICIONALES:
- Validación automática de archivos de entrada
- Conversión masiva TXT → DZN
- Guardado de resultados
- Cancelación de ejecución
- Limpiar pestañas de resultados

ARCHIVOS DE PRUEBA:
Los archivos de prueba se encuentran en el directorio DatosProyecto/
Las instancias propias del equipo están en MisInstancias/
